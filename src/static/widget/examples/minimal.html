Minimale Einbettung

Kompakte Variante ohne Statusbar und Controls — z.B. für Sidebars oder bestehende Formulare.

Ihre Anwendung

Dies simuliert eine bestehende Webanwendung. Die Metadata Agent Webkomponente ist in der rechten Sidebar eingebettet — kompakt, rahmenlos, ohne eigene Controls.

Die Komponente passt sich an den verfügbaren Platz an und kann per JavaScript gesteuert werden. Events wie metadataChange informieren die Host-Anwendung über Änderungen.
